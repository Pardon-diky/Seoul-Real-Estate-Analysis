### 서울시 부동산 투자 가성비 지표(상승률 vs 거래량) 분석

# 문제 정의
많은 투자자가 브랜드나, 인지도에 의존하여 부동산을 투자하지만, 본 프로젝트는 객관적인 실거래 지표를 통해 저평가된 지역을 찾는 분석적 접근을 목적으로 한다. 즉, 단순히 매매 가격이 높은 지역을 발굴하는 것이 아닌 분석을 통해 투자 가성비가 높은 지역을 찾는 것을 목적으로 한다. 
1. 가격은 낮지만 매매가가 빠르게 오르는 지역은 어디인가?
2. 거래량이 많고 나중에 팔기 쉬운 지역은 어디인가?

# 사용 데이터
- 공공데이터 포털 - 국토교통부 실거래가 자료
- 서울시 법정동 코드

# 기술 스택
- python script(공공데이터 포털 API 호출 및 데이터 수집)
- Hadoop HDFS
- Spark SQL
- 수집 및 시각화 : Python을 사용하여 데이터를 가져오고, 시각화

# 구현 계획
1. 데이터 수집 : Python을 사용하여 서울시 부동산 데이터를 다운로드 하여 HDFS에 저장
2. 데이터 정제 : Spark를 사용하여 콤마(,)등을 지우고, 분석하기 좋은 형태로 깔끔하게 정리
3. 데이터 분석 : Hive 와 Spark SQL을 이용하여 분석한다
- 거래량 순위: 최근 사람들이 가장 많이 사고판 지역은?
- 가성비 지역: 평당 가격은 낮지만 상승 폭은 큰 지역은?
- 신축 효과: 지은 지 얼마 안 된 아파트가 가격이 더 잘 오르는가?
